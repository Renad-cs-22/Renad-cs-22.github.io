# Personal Portfolio Website

Welcome to the repository of my personal portfolio website. This project serves as a comprehensive showcase of my skills, academic achievements, and software projects in Computer Science and Cybersecurity.

## 🚀 Live Demo
You can visit the live website here: [https://renad-cs-22.github.io](https://renad-cs-22.github.io)

## 🛠️ Features & Sections
- **Hero Section:** A brief, impactful introduction about my background and technical focus.
- **About Me:** Insights into my passion for computing, problem-solving, and continuous learning.
- **Technical Skills:** A categorized display of programming languages, tools, and frameworks.
- **Projects Showcase:** Highlights of my top development and automation projects with direct links to their repositories.
- **Contact Info:** Direct links to reach out via Email, LinkedIn, or GitHub.

## 💻 Technologies Used
- **HTML5:** For structuring the website content with semantic elements.
- **CSS3:** For responsive styling, modern layouts (Flexbox/Grid), and smooth transitions.
- **Font Awesome:** For clean, scalable vector icons.
- **GitHub Pages:** For fast, secure, and free hosting.
